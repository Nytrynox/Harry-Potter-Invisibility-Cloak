# Invisibility Cloak Effect

Real-time video effect that makes a colored cloak appear invisible by replacing it with the background, inspired by Harry Potter.

## Features

- Color-based segmentation for cloak detection
- Background capture and real-time substitution
- HSV color range tuning for different cloak colors
- Webcam and video file input support

## Tech Stack

Python, OpenCV, NumPy

## Getting Started

### Prerequisites

- Python 3.8+
- pip

### Installation

```bash
git clone https://github.com/karthik-idikuda/Harry-Potter-Invisibility-Cloak.git
cd Harry-Potter-Invisibility-Cloak
pip install opencv-python numpy
```

### Usage

```bash
python cloak.py
```
